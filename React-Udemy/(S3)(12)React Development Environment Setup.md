### ‚öôÔ∏è [[VS Code]] Configuration:

- **Essential Extensions**:
    
    - `ESLint`: Lints code and helps spot errors or bad practices [[ESLint]]

    - `ESLint` run mode: Set to `onSave` instead of `onType`
